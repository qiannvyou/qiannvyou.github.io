                          一、教学目的

                          通过公共体育足球选项课教学，使学生具备一定的足球基本技术, 了解足球发展的历史和最新的足球竞赛规则。并达到如下目标：
                            （一）基本了解足球发展概况，掌握足球的基本技术，教养学生对足球兴趣。
                            （二）提高足球的基本技术，提高学足球战术意识，身体素质得到加强。
                            （三）基本掌握几种战术配合，了解足球战术，基本掌握足球裁判法，提高足球欣赏水平。

                          二、重点与难点

                          （一）重点：运球、运球过人、踢球、头顶球、抢断球、掷界外球等足球基本技术。

                          （二）难点：运球过人、踢球以及足球技术的合理运用。

                          三、基本内容与要求

                            理论部分

                          第1章 足球运动概述

                          1.1 足球运动的起源

                          1.2 现代足球运动的发展

                          1.3 女子足球运动的发展

                          1.4 现代足球运动的发展趋势

                          （一）教学目的：使学生基本了解足球运动的起源，并了解足球运动未来发展趋势以及了解中国的足球运动发展过程。

                          （二）重点与难点：足球运动的起源与发展。

                          （三）基本内容：足球运动发展简述；足球运动的基本规律和特点；女子足球运动的发展；世界足球运动的发展趋势。

                            技术部分：

                          第2章 足球基本技术

                          2.1 足球技术的基本概念及其分类

                          2.2 运球与运球过人技术

                          2.3 踢球技术

                          2.4 接球技术

                          2.5 头顶球技术

                          2.6 抢断球技术

                          2.7 掷界外球技术

                          2.8 守门员技术

                          第3章 足球教学比赛

                          3.1 比赛阵型

                          3.2 进攻战术

                          （一）教学目的：了解足球技术的基本概念及分类，掌握运球、运球过人、踢球、接球、头顶球、抢断球、掷界外球等足球基本技术。

                          （二）重点与难点: 足球的基本技术的学习，足球的踢、接球基本技术的学习。

                          （三）基本内容：足球比赛的各种阵型；足球比赛中常用的各种进攻战术。

                          第3章  身体素质——中长跑

                          3.1 起跑和起跑后的加速

                          3.2 途中跑、终点跑

                          3.3 中长跑的呼吸

                          3.4 全程跑：1000米（男生）、800米（女生）

                          （一）教学目的：通过教学使学生掌握起跑和起跑后的加速、途中跑、终点跑等中长跑的基本技术，学会正确的呼吸方式，发展耐力素质，进一步增强心肺功能。

                          （二）重点与难点：途中跑技术；中长跑的呼吸。

                          （三）基本内容：起跑和起跑后的加速、途中跑、终点跑等基本技术；全程跑。

                          五、成绩考核：

                          （一）考核方式：现场测试。

                          综合成绩＝平时成绩×30%＋考试成绩×70%，考试成绩＝专项成绩×70%＋身体素质成绩×30%。 每一部分都有定量或定性的评分标准，其中身体素质为每学期的必考项目，身体素质考试内容为男子1000米、女子800米。 根据评分标准计算学期体育课程成绩，成绩按五级制，分为优秀、良好、中等、及格、不及格， 90分以上为优秀，80分-89分为良好，70分-79分为中等，60分-69分为及格，60分以下为不及格。

                          （二）考试内容：足球基本技术（从踢远和颠球中任意选择一项）、身体素质（男子1000米、女子800米）。

                          （三）评分标准：

                          1.踢远

                          （1）方法：用脚背内侧踢球方法踢定位球，左、右脚不限，每人可踢三次，取三次中的最好成绩。

                          （2）规则：球的落点必须在10米宽的区域内，否则为无效。

                          （3）评分标准：

                          注：踢远20米至24米为60分，25米至29米为70分，以此类推。 

                          2.颠球

                          （1）方法：受测者用下肢，胸部，头颠球。

                          （2）规则；颠球高度不限。颠球过程中球落地或手背触球时颠球即停, 考试即止。 受测者用两脚, 大腿部位颠球时, 两脚必须交替进行. 单脚或大腿连续颠球,颠球不予计算累计数.
